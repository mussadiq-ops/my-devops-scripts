# my-devops-scripts

A collection of DevOps shell scripts for automation and system management.

## Scripts
- hello.sh      — prints a greeting message
- sysinfo.sh    — displays system info (host, OS, uptime)
- cleanup.sh    — cleans up temporary files
- backup.sh     — creates a timestamped backup
- monitor.sh    — shows CPU, RAM and disk usage

## Tech Stack
- Shell / Bash
- Git + GitHub
- WSL (Windows Subsystem for Linux)
